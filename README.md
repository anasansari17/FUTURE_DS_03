# Marketing Funnel & Conversion Performance Dashboard

## Project Overview

This project analyzes an e-commerce marketing funnel to understand how users move from **product views → cart additions → purchases**.

The dashboard identifies conversion drop-offs, high-performing brands, and revenue-driving product categories.

## Tools Used

* Power BI
* Data Cleaning
* DAX Measures
* Data Visualization

## Key Metrics

* Total Visitors: 49K
* Total Leads (Add-to-Cart): 578
* Total Customers (Purchases): 751
* Overall Conversion Rate: 1.54%

## Key Insights

* The largest drop-off occurs between product views and cart additions.
* Samsung and Apple generate the highest number of purchases.
* Electronics categories drive the majority of revenue.
